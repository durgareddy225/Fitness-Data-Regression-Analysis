# Fitness Data Regression Analysis
## Overview
This project focuses on analyzing fitness-related data using a Random Forest Regression model. The goal is to understand how different workout and personal attributes influence calorie burn and evaluate the model’s performance using regression metrics.

---

## Features
- Data preprocessing and cleaning
- Handling missing values
- Encoding categorical variables
- Training a Random Forest Regression model
- Model evaluation using:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - Root Mean Squared Error (RMSE)
  - R² Score

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

---

## Dataset Information
- **Source:** Kaggle – [Gym Members Exercise Dataset](https://www.kaggle.com/datasets/valakhorasani/gym-members-exercise-dataset)
- The dataset contains fitness and workout-related information used to analyze calorie burn and exercise patterns.

---

## Model Details
- **Algorithm:** Random Forest Regression
- **Type:** Supervised Learning
- **Task:** Regression Analysis

---

## Evaluation Metrics
The model performance is evaluated using:
- **MAE** – Measures average prediction error
- **MSE** – Average squared error
- **RMSE** – Penalizes large errors
- **R² Score** – Indicates how well the model fits the data
